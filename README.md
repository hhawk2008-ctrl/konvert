# Ясновидение. Основы восприятия информации — лендинг курса

Одностраничный лендинг для однодневного практического курса Сергея Ратнера
«Ясновидение. Основы восприятия информации».

## Содержание

- `index.html` — разметка страницы (УТП, описание, для кого, чему научитесь,
  что будете уметь, результат, автор, форма записи).
- `styles.css` — стили (тёмная элегантная тема, адаптивная вёрстка).
- `script.js` — анимация появления секций и обработка формы записи.

## Как открыть

Это статический сайт без сборки. Достаточно открыть `index.html` в браузере
или запустить локальный сервер:

```bash
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## Форма записи

Форма пока работает только на стороне клиента: заявки сохраняются в
`localStorage` и показывают подтверждение. Для приёма реальных заявок
подключите бэкенд или сервис форм в `script.js`.
